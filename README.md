# Cognifyz-web-dev-task3

💻 Tech Gallery

A modern and interactive Tech Gallery Web Application built using HTML, CSS, and JavaScript.
This project showcases a responsive image gallery with animations, popup preview, slideshow, and dark/light mode support.

---

🚀 Features

- 🖼️ Responsive Image Gallery (Grid Layout)
- 🔍 Image Popup Preview on Click
- 🎞️ Auto Slideshow with Smooth Transitions
- 🌙 Dark / Light Mode Toggle
- ✨ CSS Animations & Hover Effects
- 📱 Fully Responsive Design
- 🧊 Glassmorphism UI Styling

---

🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (DOM Manipulation)
